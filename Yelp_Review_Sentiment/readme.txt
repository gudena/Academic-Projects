We have done our project using Databricks and pre-processing in SAS using python. Following are the steps to run our project:

2) Pre-processed data should be uploaded into Databricks.

3) Import the notebook present in databricks url attached.

4) Create the cluster.

5) Attach the notebook to cluster and run.
